STEP 1: Install Python Dependencies
────────────────────────────────────
Windows:
  pip install -r requirements_task2.txt

CentOS/Linux:
  pip3 install -r requirements_task2.txt

Or manually:
  pip install fastapi uvicorn pydantic requests streamlit

STEP 2: Install Ollama
───────────────────────
Windows:
  1. Download: https://ollama.com/download
  2. Run installer
  3. Ollama auto-starts

CentOS/Linux:
  curl -fsSL https://ollama.com/install.sh | sh

STEP 3: Download AI Model
──────────────────────────
Both platforms:
  ollama pull llama3.2:1b

  (Downloads ~1.3GB, wait 2-3 minutes)

Verify:
  ollama list

"Backend API not running"
  → Make sure Terminal 1 is running python app.py

"Module not found"
  → Run: pip install -r requirements_task2.txt

"Ollama not connected"
  → Windows: Check system tray
  → Linux: sudo systemctl start ollama

"Port already in use"
  → Windows: Restart computer
  → Linux: pkill -f "python3 app.py"

"Model not found"
  → Run: ollama pull llama3.2:1b
