===============================================
OLLAMA SETUP GUIDE - Recipe Chatbot
===============================================

For WINDOWS:
-----------
1. Download Ollama from: https://ollama.com/download
2. Run OllamaSetup.exe and follow the installer
3. Ollama will start automatically
4. Open Command Prompt and run:
   ollama pull llama3.2:1b
5. Wait for download to complete (~1.3GB)
6. Verify: ollama list

For LINUX:
---------
1. Open terminal and run:
   curl -fsSL https://ollama.com/install.sh | sh
2. Download the model:
   ollama pull llama3.2:1b
3. Wait for download to complete (~1.3GB)
4. Verify: ollama list

For MAC:
-------
1. Open terminal and run:
   curl -fsSL https://ollama.com/install.sh | sh
2. Download the model:
   ollama pull llama3.2:1b
3. Wait for download to complete (~1.3GB)
4. Verify: ollama list

VERIFICATION:
------------
Run: ollama list
You should see: llama3.2:1b

TROUBLESHOOTING:
---------------
If "ollama: command not found":
- Windows: Restart your computer
- Linux: Run "sudo systemctl start ollama"
- Mac: Ollama should auto-start

If model not downloading:
- Check internet connection
- Try: ollama pull llama3.2:1b --verbose

===============================================
After Ollama setup, proceed with:
1. pip install -r requirements_task2.txt
2. python app.py (Terminal 1)
3. streamlit run streamlit_ui.py (Terminal 2)
===============================================
